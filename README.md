# SardineClaw
## 🎯简介
- 一个全python的AI Agent项目
- GUI使用Flet
- LLM所有能力依赖SKILL技能,不乱跑,不瞎想

## 📂项目目录
sardine-claw/

    ├── skills/                         # 技能/模块目录
    
    ├── agent.py                        # 智能体核心实现
    
    ├── agents_conf.yaml                # 智能体配置文件
    
    ├── AI全流程.jpg                     # AI 流程示意图
    
    ├── const.py                        # 常量定义文件,主要是界面控制和LLM提示词
    
    ├── gui.py                          # 图形界面入口
    
    ├── llm.yaml                        # 大语言模型配置文件
    
    ├── main.py                         # 程序主入口文件
    
    ├── msg.py                          # 消息处理模块
    
    ├── README.md                       # 项目说明文档
    
    └── requirements.txt                # 项目依赖包列表
